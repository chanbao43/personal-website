# 个人网站项目

这是一个现代化的个人网站项目，采用纯 HTML/CSS/JavaScript 实现。

## 功能特点

- 现代简约的设计风格
- 响应式布局，支持移动端
- 个人简介展示
- 作品集展示
- AI 对话界面（基于 DeepSeek API）
- 支持深色模式
- 支持多种格式内容：视频、图片、文字、Markdown

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- DeepSeek API

## 本地开发

1. 克隆仓库

```bash
git clone https://github.com/chanbao43/personal-website.git
cd personal-website
```

2. 打开项目

- 使用浏览器直接打开 `index.html`
- 或使用本地服务器（推荐）

## 部署

项目可以部署在任何静态网站托管服务上，如：

- GitHub Pages
- Vercel
- Netlify
